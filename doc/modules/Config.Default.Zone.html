Tables

marker
Marker Settings Settings for a marker, including its appearance and behavior.

Fields:

  • type int Marker type (e.g., 27).
  • direction vector3 Direction of the marker.
  • rotation vector3 Rotation of the marker.
  • scale vector3 Scale of the marker.
  • color table Marker color settings (RGBA).
    • r int Red component (0-255).
    • g int Green component (0-255).
    • b int Blue component (0-255).
    • a int Alpha component (0-255).
  • animate bool Whether the marker animates.
  • faceCam bool Whether the marker faces the camera.
